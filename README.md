# GymMe

## Project Description
GymMe is an ASP.NET Web Forms eâ€‘commerce application for managing and selling gym supplements. It provides customer-facing shopping and profile features and an admin panel for inventory, transaction and user management, with reporting support.

## Features
- Product browsing with supplement details (type, price, expiry)
- Shopping cart: add, update, remove items
- Checkout and transaction processing with history and detail views
- User authentication and profile management
- Admin CRUD for supplements and supplement types
- Inventory and expiry tracking
- Transaction reporting (Crystal Reports)
- Server-side validation and logging

## Technology Stack
- Backend: ASP.NET (.NET Framework 4.7.2), C#
- ORM / Data: Entity Framework 6.x, SQL Server (LocalDB)
- Frontend: ASP.NET Web Forms (ASPX)
- Reporting: Crystal Reports 13
- Logging: log4net
- Tooling: Visual Studio, Microsoft CodeDom providers
